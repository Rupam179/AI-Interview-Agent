What is the difference between a list and a tuple in Python? When would you use each?
🎙️

📝 Transcribed Answer
A list is mutable — you can add, remove, or change elements after creation. A tuple is immutable — once created, it cannot be changed. I use lists when I need to modify data like storing user inputs, and tuples for fixed data like coordinates or database records where immutability is preferred for safety.

About This Project
AI Interview Agent — Final Year ESD Project
🤖 Project Overview
AI Interview Agent is a fully functional AI-powered Interview Preparation Platform built as a Final Year Project for B.Tech CSE (AI & ML) at Techno India University. The platform leverages Google Gemini 1.5 Flash to help students prepare through intelligent question generation, resume analysis, voice interviews, and personalized feedback.
